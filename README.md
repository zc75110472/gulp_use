# gulp_use
gulp的使用

## gulp 压缩 css html js的构建工具
### 首先 gulp是基于 node环境  
# npm install gulp -g
```
  npm install gulp
  npm install gulp-uglify    // 压缩 js的
  npm install gulp-concat    // 合并js css用
  npm install gulp-minfy-css   // 压缩css
  npm install gulp-minfy-html   // 压缩html
```
# 详情请看 gulpfile.js
